-Pertama adalah membuat file index yang di dalamnya kita menentukan port yang di pakai oleh express,kemudian menyiapkan koneksi yang akan digunakan oleh mongo db,kemudian menghidupkan fungsi cors.
-membuat schema mongo db untuk file yang akan diterima oleh mongo db nantinya.
-membuat controler untuk schema yang mana dia akan berisi pekerjaan yang kira ingin kan misalnya edidt,delete,create dll, dan di dalamnya tentu kita akan mengimport schema model yang sudah kita buat sebelumnya.
-selanjutkan yaitu membuat route yang akan dilalui oleh pekerjaan yang telah kita buat sebelumnya dan kita import pada route yang kita buat tersebut,misalnya jika dia adalah edit,maka kita akan mengarahkannya apada route/rute yang benar,dan tentu route tersebut akan kita import pada index.js agar app.use (methode express) dapat berjalan untuk route tersebut.
-ketika kita membuat sebuah route (/post/:id) maka kita dapat mengakses dengan req.param atau req.body,dengan req.body kita dapat megirrimkan data yang ingin kita simpan ke database melalui melalui url yang telah kita buat,dan tentu itu akan menyesuaikan dengan model schema yang sudah kita buat sebelumnya.
-pada updatePost kita akan mengupdate post dengan menggunakan method updateOne yang mana akan membawa 2 parameter,yaitu parameter pertama adalah post/:id yang akan kita update dan parameter yang kedua berisi set req.body karena akan mengambil data melalui itu.
-pada bagian client kita akan membuat file di dalam ./src/components dengan nama post list yang mana akan mengambil data dari api menggunakan axios,axios sendiri digunakan untuk mempermudah kita mengambil data menggunakanan api,dalam langkahnya kita akan mengunakan useState untuk menyimpan data terbaru dari api kedalam posts(useState),dan dengan mengunakan useEffect yang mana didalamnya kita akan memasukkan method getPosts yang sudah kita buat yang mana berisi pengambilan data menggunakan api,ketika halaman dirender maka useEffectakan memanggil getPosts dan setPosts yang ada di dalam method getPosts akan memasukkan data kedalam posts(useState) dan function component PostList akan mereturn dengan menampilkan data posts(useState) dengan cara melooping data menggunakan method.map
-pada sisi client kita akan menambahkan UI untuk bisa menambahkan post ke sisi server,sama dengan sebelumnya kita bisa melkukan request dengan axios.post yamng memiliki 2 parameter yang pertama adalah endpoint yang akan kita tuju dan yang kedua adalah isi dari useState yang sudah kita buat,useState itu sendiri akan diisi dengan form yang sudah di sediakan yang mana akan mengambil data yang dimasukkan dengan menggunakan target.value yang mana masing-masing form akan memiliki value yang diisi dengan useStateyang sudah dibuat.
-useParams digunakan untuk dapat mengambil nilai berupa object yang dikirim melalui url
